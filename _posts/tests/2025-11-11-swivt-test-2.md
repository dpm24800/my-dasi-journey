### 18. Name Formatter :Ask the user to enter their full name and print:
    - First name 
    - Last name 
    - Initials 
    - Total number of characters (excluding spaces)

```py
name = input("Enter your Name")

list = name.split(" ")
initials = ""
word_count = 0

if len(list) == 2:
    print("First Name:", list[0])
    print("Last Name:", list[1])
    word_count = len(list[0]) + len(list[1])
elif len(list) == 3:
    print("First Name:", list[0])
    print("Middle Name:", list[1])
    print("Last Name:", list[2])
    word_count = len(list[0]) + len(list[1]) + len(list[2])

for i in list:
    initials += i[0]

print("Initials: ", initials)
print("\nWord count (excluding space): ", word_count)
```

```
First Name: Manoj
Last Name: Lamsal
Initials:  ML

Word count (excluding space):  11
```

### 26.  Find and Replace in List Ask the user for a word list and replace all occurrences of "bad" with "good".
```py
# manoj, 10, bad, sita, 20, nice, niraj, 20, bad, hari, 40, awesome, niraj, 25, bad
words = input("Eneter a list with seperator (, ): ")

list = words.split(", ")

print("original:", list)

print("replaced: ", list)
```
